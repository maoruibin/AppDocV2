# 隐私政策

##  1.我们会收集什么信息？

《点亮》非常重视用户的隐私保护，它绝不会收集您的任何隐私内容和信息。

##  2.权限申请说明

- 存储权限：仅用于备份和导出功能，除此之外《点亮》不会私自访问或修改您外置存储中的任何文件。
- 网络权限：仅用于在您允许的情况下将数据备份到 WebDAV 服务器。

## 3.信息的披露

《点亮》不会收集您的个人信息，不会他人分享这些信息，亦不会将其出售、出租以及其他方式分发个他人。

##  4.数据安全

《点亮》将致力于保护您的数据信息安全,它只会把您所填写的内容保存在手机本地数据库中。

##  5.未成年人信息及使用

《点亮》非常重视对未成年人个人信息的保护,我们希望未成年人仅在接受其父母或法定监护人指导、监督并经其同意的前提下使用。
