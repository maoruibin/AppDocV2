# 关于  

PicPlus 是一个图床上传 APP，可以帮你上传手机图片到图床，支持七牛云、阿里云、又拍云等主流图床，同时还支持配置 GitHub、码云为个人图床。

上传时支持自动压缩图片，上传后自动生成 Markdown、Html 格式链接。

## 开发初衷
对于很多使用 Markdown 的写作爱好者，插图是一个很常见的操作。如果你在 PC 上写文章，可以使用 iPic、PicGo 等工具上传图片到图床，但是如果你在手机上写笔记，或者 PC 上写作时，需要手机中拍摄的图片时，上传图片就会变的麻烦。

这时，如果有一个手机软件，可以方便的把图片上传到图床，并生成一个 Markdown 图片链接，就会大大提高写作者的生产效率，PicPlus  就是基于此场景开发一个手机图床上传软件，效果如下所示：

除了在 APP 中可以上传图片，**也可以在系统任何图片应用中，选择系统分享，然后传图到图床**，具体如下所示：

PicPlus 支持市面上主流的云存储服务，如阿里云、七牛云、又拍云等，还支持配置 GitHub、码云为个人图床：



![](https://cdn.nlark.com/yuque/0/2020/jpeg/159409/1588747118450-3d8cd9f5-98a2-46b6-b309-16ddd4c0833b.jpeg)



更多功能特性如下所示：

+ 支持 Github、码云为图床
+ 拍照上传图床
+ 上传前自动压缩
+ 可删除已上传的远程图片
+ 批量上传相册图片
+ 自动格式化图片链接为 Markdown、HTML 格式
+ 支持通过系统分享上传图片
+ WebDAV 数据备份
+ 支持自动复制链接
+ 支持设置 Markdown 格式的图片标题
+ 支持数据导出、导入，自动备份本地数据
