# 配置 七牛云 为个人图床

## 写在前面


使用七牛云做图床需要配合一个**已经备案的域名**，所以如果你没有自己的域名，或者域名没有备案，则不建议使用，也无法使用七牛云。建议你使用**码云、GitHub或者 SM.MS **作为自己的图床、

> 这份教程略去了注册以及域名解析相关的教程，如果对这方面有需要的同学，可以查看一位热心的网友为 PicPlus 写的[详细版七牛云配置教程](https://www.yuque.com/gudong-osksb/twgz5k/aq3fd3)
>

## 配置七牛云说明


简单点说，配置七牛云需要四个参数



+ 密钥 AK（AccessKey）
+ 密钥 SK （SecretKey）
+ 存储空间名称
+ 自定义的备案域名



只有配置了前两个参数，PicPlus 才有权限去操作七牛云。



只有配置了第三个参数，PicPlus 才知道要把图片存储在什么位置。



## 获取 AK/SK


登录七牛云，进入自己的控制台，控制台入口在主页右上角，如下所示：



![](https://cdn.nlark.com/yuque/0/2020/png/159409/1583672941265-7b6c032d-5d03-4a33-8b65-a2ab6dd6492c.png)



进入控制台后，点击右上角的个人头像，选择`密钥管理`去查看密钥，如下所示：



![](https://cdn.nlark.com/yuque/0/2020/png/159409/1583672940950-f29c3793-d9ca-487b-83fa-8222bf5a92e9.png)



点击进入密钥管理页面，如果你还没有密钥，需要点击`创建密钥`，



![](https://cdn.nlark.com/yuque/0/2020/png/159409/1583672940923-9ee6eaa3-f28a-454d-b460-4140516e68da.png)



创建完毕后，发送 AK、SK到手机，



![](https://cdn.nlark.com/yuque/0/2020/png/159409/1583672940911-47c52ca5-b8ee-441a-bb1e-e3ae20100fa8.png)



AK、SK创建完毕。



## 存储空间名称


选择七牛云左侧菜单栏中的`对象存储`一项。



![](https://cdn.nlark.com/yuque/0/2020/png/159409/1583672940916-24d52594-e7f5-4afc-9a25-00e8b1d5066f.png)



选择空间管理：



![](https://cdn.nlark.com/yuque/0/2020/png/159409/1583672941106-f7cadc28-39e8-42cd-93ff-4cf05fb9be90.png)



如果还没有空间，则选择`新建空间`



![](https://cdn.nlark.com/yuque/0/2020/png/159409/1583672940957-65230875-dc26-46d5-aa54-b6bb15dba1c1.png)



点击新建后，输入空间名，同时选择访问控制为`公开`，否则别人无法访问你的图片。



![](https://cdn.nlark.com/yuque/0/2020/png/159409/1583672941216-98039802-8a3f-4ce5-aaca-2c466a972c28.png)



确认后，空间已经创建好。



至此前三个参数都已经获取完毕，打开 PicPlus 七牛云配置界面



## 配置 PicPlus


![](https://cdn.nlark.com/yuque/0/2020/jpeg/159409/1583672946569-6851e252-c656-4fc2-8109-7b493ad65009.jpeg)



将上面生成的信息拷贝到对应的输入框。



最后一项需要输入自己的域名



> Note: **请以 http 或者 https 开头**
>



配置好之后，可前往 PicPlus 主页进行验证。
