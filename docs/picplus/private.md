# 隐私政策

> 更新日期：2025 年 10 月 28 日  
> 生效日期：2025 年 10 月 28 日

欢迎使用咕咚云图（以下简称“我们”或“咕咚云图”）。我们非常重视您的隐私与数据安全。本隐私政策旨在向您说明在使用咕咚云图服务过程中，我们如何收集、使用、存储与共享您的信息。除非另有说明，本政策适用于咕咚云图 App 及其提供的相关功能与服务。

一、我们如何收集与使用信息

1. 设备信息与应用信息（必需）  
为保障应用的基本运行、安全与问题排查，我们会收集您的设备型号、系统版本、系统语言、屏幕尺寸、应用版本号、崩溃日志、异常日志与性能信息等。我们不会收集设备唯一标识（如 IMEI、IMSI、OAID、Android ID 等）。上述信息仅用于：
- 确保应用正常运行与功能可用；
- 定位与修复崩溃、卡顿、网络错误等问题；
- 统计匿名使用情况以优化产品体验。

2. 账号与身份信息（如适用）  
咕咚云图提供可选的账号功能（例如会员权益与偏好同步，如适用）。我们不要求提供真实姓名、身份证号等高敏信息；可能采集账号标识（如 UID）、登录凭据（token）、会员状态等用于账号管理与权益校验。您可在“关于”页面最底部通过“注销账号”入口发起注销，详见“十、账号注销流程”。

3. 图床配置与上传数据（本地处理为主）  
当您使用咕咚云图配置图床（如 S3/COS/OSS/又拍云/七牛等）并上传图片时：
- 配置项（例如 AccessKeyId、AccessKeySecret、Endpoint、Bucket、Region、文件路径规则等）保存在您本地设备的偏好或本地数据库中，不会上传到我们的服务器；
- 凭据本身按当前实现未进行额外加密（例如直接以明文或系统安全存储方案保存），我们强烈建议您妥善保管设备并设置系统级锁屏与密码；
- 上传的图片、生成的链接与历史记录仅在本地保存，用于便于您查看与管理，不会传输至我们的服务器。

4. 第三方 SDK 与服务  
为向您提供云存储/图床上传能力，我们可能会在 App 中集成以下第三方 SDK 或通过其官方 API 进行上传：
- 七牛云（Qiniu Cloud）：用于对象存储与上传；
- 腾讯云 COS（不含 Beacon 安全组件）：用于对象存储与上传；
- 阿里云 OSS：用于对象存储与上传；
- 又拍云（Upyun）：用于对象存储与上传。
上述 SDK/服务由对应的第三方提供与维护，其收集与处理数据的方式请以各自的隐私政策与合规声明为准。我们不使用与安全审计无关的统计、广告或追踪 SDK。

二、本地数据存储与安全

- 我们在本地存储您的图床配置、上传记录与偏好设置，存储位置可能包括 Android 的 SharedPreferences、应用私有目录、以及本地数据库；
- 我们不将上述数据同步到任何第三方服务器；
- 凭据类信息可能不具备额外加密，请勿在共享设备上保存敏感配置，并为设备启用安全措施（如指纹/面容/锁屏密码）；
- 若您卸载应用，上述本地数据将随系统清理一并删除（除非系统或设备厂商保留了备份数据）。

三、云端数据处理与跨境传输

- 当您使用第三方云存储或图床服务上传图片时，图片与相关元数据将直接传输至您选择的服务提供商（例如阿里云、腾讯云、七牛云、又拍云等），这些数据不经过我们的服务器；
- 数据可能根据服务提供商的地域设定在境内或境外存储与处理；
- 请您仔细阅读并理解所选服务提供商的隐私政策、服务条款与合规承诺；
- 我们不对第三方服务的数据处理行为承担责任，但会尽力提醒您关注其合规性与安全性。

四、Cookies 与网页组件

- 咕咚云图为原生 App，不使用 Cookie 进行用户追踪；
- 如应用内嵌少量网页组件（例如文档说明链接），Cookie 的使用将遵循该页面所在服务的政策；
- 我们不进行跨站跟踪，不构建广告或营销画像。

五、权限使用说明

为实现图片选择、拍摄与上传等核心功能，我们可能申请以下系统权限：
- 存储权限（读取与写入）：用于选择本地图片、保存上传日志/缓存；
- 媒体库访问权限（Android 13+ 等系统）：用于访问图片/视频媒体文件；
- 相机权限：用于拍摄并上传图片（仅在您使用拍照相关功能时申请与使用）；
- 网络权限：用于与您选择的图床/云存储服务进行通信；
- 网络状态与 Wi-Fi 状态：用于判断网络可用性、优化上传体验。
我们不会申请与功能无关的权限（例如麦克风、通话、短信、读取通讯录等）。

六、您的权利

您对自己的数据享有以下权利：
- 访问与导出：您可在 App 中查看历史记录与链接列表；
- 更正与删除：您可随时修改图床配置或清除本地记录；
- 权限管理：您可在系统设置中随时收回已授权的权限；
- 注销与卸载：卸载 App 后，本地数据将被系统清理。

七、未成年人保护

咕咚云图面向成人用户，不建议未成年人使用。如未成年人确需使用，请在监护人同意与指导下进行。我们尽力保护未成年人的隐私与数据安全。

八、政策更新

当我们对隐私政策进行重大调整时，会在应用内或官网以显著方式提示并更新版本日期。继续使用即视为您已阅读并同意更新后的政策。若您不同意，请立即停止使用并卸载应用。

十、账号注销流程

- 入口位置：在 App “关于”页面最底部提供“注销账号”入口；
- 操作流程：点击“注销账号”→弹出确认提示→确认后立即执行注销；
- 注销范围：清除账号标识、登录态与会员状态，删除我们服务端的您的账号数据；同时清理本地与账号相关的数据；
- 不受影响的数据：账号注销不影响您在第三方图床/云存储服务（如 S3/COS/OSS/又拍云/七牛等）的账号与已上传内容；本地图床配置与上传记录默认不随账号注销自动删除，以避免影响已生成链接，您可在设置中手动清理；
- 数据处理与留存：注销通常即时生效；为符合法律法规或处理争议的需要，我们可能在法定期限内保留必要的日志信息；
- 注销后果：注销不可恢复，若需继续使用，可重新注册或登录；如涉及付费会员，注销后权益失效不可恢复。
