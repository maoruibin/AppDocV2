## 详细特权介绍

### 1. 无限资产记录

**基础版**：限制最多记录50条资产
**PRO版**：无资产记录数量限制，可随心记录所有资产

无论您拥有多少件收藏品、多少项投资、多少笔应收款项，PRO版都能满足您的需求，帮助您全面管理资产，不再因数量限制而烦恼。

### 2. 自动云备份（计划中）

**基础版**：仅支持手动本地备份
**PRO版**：支持自动云端备份，定期保存数据到云端

PRO版提供自动云备份功能，可设置每日、每周或每月自动将数据备份到云端，即使设备丢失或损坏，您的数据也安全无忧。云备份支持最近7次备份记录，可随时恢复任意一次备份数据。

### 3. 高级数据统计

**基础版**：仅提供基础统计图表
**PRO版**：提供深度资产分析和财富报告

PRO版提供更专业的资产分析功能：
- 资产变化趋势图：直观展示资产价值变化
- 分类占比分析：细化各类资产占比和结构
- 定期财富报告：每月/季度/年度资产报告
- 自定义统计维度：按需选择统计指标和维度


### 4. 优先技术支持

**基础版**：普通客户支持
**PRO版**：优先响应的技术支持服务

PRO用户可获得优先级更高的技术支持服务，问题和反馈会得到更快速的响应和解决。同时，PRO用户的功能建议也将优先被考虑纳入产品更新计划。
