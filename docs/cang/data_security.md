# 数据安全与备份

仓咚咚高度重视用户数据安全和隐私保护，采用多种技术手段确保您的资产数据安全可靠。本文详细介绍仓咚咚的数据安全措施和备份方案。

## 数据安全保障

### 本地存储优先

仓咚咚采用"本地存储优先"的设计理念，您的核心资产数据默认存储在设备本地：

- 所有资产信息优先保存在设备本地数据库
- 资产照片和附件保存在设备本地加密存储区
- 只有选择云备份功能时，数据才会上传到云端
- 可完全离线使用，不强制要求网络连接

### 数据加密

仓咚咚对敏感数据采取严格的加密措施：

- 本地数据库采用业界标准的 SQLCipher 加密，防止设备被盗后数据泄露
- 云端备份数据采用端到端加密，即使在传输和存储过程中也无法被第三方解密
- 资产金额等敏感信息可选择额外加密，增加一层保护

### 隐私保护措施

为保护您的隐私，仓咚咚提供多种隐私保护功能：

- 指纹/面容验证：支持使用设备生物识别验证身份，防止未授权访问
- 应用锁定：可设置应用启动密码，增加一层安全保护
- 隐藏资产价值：可选择在资产列表中隐藏价值信息，防止他人窥视
- 隐私模式：一键切换到隐私模式，临时隐藏敏感资产信息

## 数据备份方案

### 本地备份（所有用户）

仓咚咚为所有用户提供本地备份功能：

1. **手动备份**
   - 进入"设置">"数据管理">"本地备份"
   - 点击"创建备份"生成加密备份文件
   - 备份文件默认保存在设备的Documents/Backups目录下
   - 支持手动分享备份文件到其他位置（如云盘、邮箱等）

2. **备份内容**
   - 所有资产数据和配置信息
   - 用户自定义分类和标签
   - 统计设置和偏好配置
   - 可选择是否包含资产图片（减小备份文件体积）

3. **备份恢复**
   - 进入"设置">"数据管理">"恢复备份"
   - 选择要恢复的备份文件
   - 可选择是否合并或覆盖现有数据

### 云端备份（PRO版功能）

PRO版用户可享受更便捷的自动云备份功能：

1. **自动备份**
   - 可设置自动备份频率：每日、每周或每月
   - 可选择仅在WiFi环境下进行备份
   - 最多保留最近7次备份记录

2. **云端安全措施**（计划中）
   - 备份数据采用AES-256加密存储
   - 云端服务器位于安全合规的数据中心
   - 定期安全审计和漏洞扫描

## 数据导入导出

除了常规备份外，仓咚咚还支持数据导入导出功能：

### 数据导出

- **导出格式**：支持CSV、Excel和专用格式导出
- **导出范围**：可选择导出全部资产或特定分类的资产
- **导出方式**：可保存到本地或直接分享到其他应用

### 数据导入

- **支持格式**：支持导入CSV、Excel文件
- **智能匹配**：自动匹配字段名称，简化导入流程
- **批量导入**：一次性导入大量资产数据
- **导入验证**：自动检查导入数据的合法性和完整性

## 数据安全最佳实践

为确保您的资产数据安全，我们建议：

1. **定期备份**：养成定期手动备份的习惯，或使用PRO版的自动备份功能
2. **多处存储**：重要备份文件保存在多个位置，避免单点故障
3. **启用生物识别**：设置指纹或面容识别，防止未授权访问
4. **保护备份密码**：妥善保管备份和恢复所需的密码
5. **及时更新**：保持应用为最新版本，获取最新的安全修复
