# 适用场景

仓咚咚作为一款全能资产管理工具，适用于多种用户场景。本文将介绍几种典型的使用场景，帮助您更好地利用仓咚咚管理自己的资产。

## 个人资产管理

### 全面了解个人财富状况

小王是一名上班族，拥有多种形式的资产，包括工资收入、银行存款、基金投资、房产和各类消费品。随着资产增多，他发现越来越难以全面掌握自己的财富状况。

使用仓咚咚后，小王可以：
- 记录所有资产，形成完整的个人资产清单
- 通过图表直观了解资产分布状况
- 追踪资产总值的变化趋势
- 发现资产配置中的潜在问题

**建议功能**：资产总览、资产分布图表、资产变化趋势图

### 追踪投资组合表现

李女士是一位积极的投资者，拥有股票、基金、债券等多种投资产品。她需要一种方便的方式来追踪这些投资的表现。

使用仓咚咚后，李女士可以：
- 分类记录各类投资产品
- 定期更新投资价值，追踪收益率
- 分析不同投资类别的表现对比
- 根据收益情况调整投资策略

**建议功能**：投资资产记录、收益率计算、投资表现分析

### 管理个人收藏品

张先生是一位收藏爱好者，拥有众多邮票、钱币和模型。他需要一个系统来管理这些收藏品，并追踪它们的价值变化。

使用仓咚咚后，张先生可以：
- 详细记录每件收藏品的信息，包括购入日期、购入价格、估值等
- 为收藏品添加照片，方便查看和展示
- 追踪收藏品价值的变化
- 合理规划收藏品的购入和出售

**建议功能**：收藏品记录、图片附件、价值变化追踪

## 家庭资产管理

### 夫妻共同财产管理

刘先生和刘太太结婚多年，拥有共同的家庭资产，包括房产、车辆、家具电器、银行账户等。他们需要一个工具来共同管理这些家庭资产。

使用仓咚咚后，这对夫妻可以：
- 建立家庭资产清单，明确每项资产的所有权
- 记录重要资产的购买信息和保修期
- 追踪家庭总资产的变化
- 规划未来的大额支出和资产购置

**建议功能**：多用户协作(PRO)、资产所有权标记、保修期提醒

### 家庭财务规划

陈家是一个四口之家，除了日常开支外，还需要为孩子教育、养老和应急情况做财务规划。

使用仓咚咚后，陈家可以：
- 全面了解家庭资产状况
- 根据资产流动性合理规划资金使用
- 设置教育金、养老金等特定目标的专项资产
- 监控家庭资产的长期增长情况

**建议功能**：资产流动性分析、目标资金设置、资产增长趋势分析

## 使用贴士

无论您属于哪种使用场景，以下建议有助于您更好地使用仓咚咚：

1. **建立科学分类**：根据自身需求设置合适的资产分类体系
2. **定期更新**：定期更新资产价值，保持数据准确性
3. **善用标签**：使用标签功能为资产添加多维属性，便于筛选
4. **记录附件**：为重要资产保存相关凭证和照片
5. **定期分析**：利用统计分析功能定期审视资产状况，及时调整
6. **注重数据安全**：开启指纹/面容验证，定期备份数据
7. **PRO升级**：如有大量资产需管理，建议升级PRO版享受更全面的功能
