# 仓咚咚

<img src="../public/img/cang_home.png" width="100%" alt="">

> 让资产一目了然

仓咚咚是一款专为个人和家庭设计的多功能资产管理应用，帮助用户全面管理各类资产，包括实物资产、金融资产、虚拟资产以及应收款项等。

通过简洁直观的界面和强大的功能，仓咚咚让资产管理变得简单高效，助您掌握财富全貌，做出更明智的理财决策。

## 核心功能

- **全方位资产分类**：实物资产、金融资产、虚拟资产、应收款项等多维度分类
- **直观的资产统计与分析**：图表化资产分布，清晰了解资产结构
- **便捷的资产管理**：快速添加、编辑和删除资产记录
- **数据安全与备份**：云端自动备份功能（PRO版本专享）
- **个性化用户体验**：支持深色模式、隐私保护等个性化设置
