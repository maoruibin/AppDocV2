# 盒子
盒子是 inBox 提供的一个功能，你可以在设置 > 功能 中找到这个功能的入口。

该功能默认关闭，需要手动打开。

打开后，在主页最顶部，会看到 inBox 名称旁边出现一个三角符号，点击后就可以看到盒子面板。

在这里，你可以选择新建盒子。

那什么是盒子？

简单说就是传统笔记软件中的分类或者文件夹，inBox 本身支持标签，可以对笔记打不同的标签，标签更加自由。

盒子则更加严格，一个笔记只能归类到一个盒子，比如建一个盒子，专门用来存放自己的密码等信息，可以叫做密码，还可以新建一个个人重要信息的盒子，专门收集一些对自己长期重要的信息。

其实以上通过标签也可以做到，但是 inBox 就是提供了一种更简单的交互，来降低使用成本。
