#  标签

标签是一种更灵活的标记笔记的形式，在笔记中，可以通过插入 # 然后再后面紧跟标签名称，这样保存笔记后，inBox 就会帮你生成一个标签，并将这个标签关联到这篇笔记。

比如写一篇笔记内容为：

> 没有什么能够阻挡，你对自由的向往。#歌词

当保存笔记后，就会自动生成一个歌词的标签，并将这个标签关联到这篇笔记，在侧边栏你可以找到这个标签。

##  多级标签

标签支持多层级，比如 #学习/技术 只需要标签中添加符号 `/` 即可建立前后的父子关系，表示前者是后者的父标签，这篇笔记也将会跟多个标签进行关联。

##  标签层级修改

inbox 笔记中，一直都可以对标签进行重命名，但是只能修改当前一级，不能支持多级修改.

所以想要整理标签体系，对原来的标签进行层级修改就很难，比如之前有一个标签 #学习/篮球，现在想要让篮球归属在 #运动标签下，也就是要改为 #学习/运动/篮球，是很麻烦的，现在只需要选中篮球标签，选择重命名，然后 将篮球改为 运动/篮球即可。

这样点击确认后，所有#篮球的标签，都会被移动到#运动的下一级，除此之外，其他场景也都可以随意通过符号 `/` 去更层级。
