# 如何导入其他笔记
inBox 支持各种平台、格式的笔记导入。

<img src="../public/img/import-1.png" width="80%" alt="">

下面以导入微信读书为例，说明如何导入笔记

## 如何导入微信读书笔记
1. 在微信读书中选择「我」页面，打开笔记模块，找到对应书籍的读书笔记，选择导出，然后复制到粘贴板
<img src="../public/img/import2.png" width="100%" alt="">

2. 打开 inBox，点击首页右上角菜单，选择导入笔记
3. 选择微信读书，打开后，点击空白区域，粘贴笔记
4. 点击开始解析，选择导入即可。
<img src="../public/img/import3.png" width="100%" alt="">
